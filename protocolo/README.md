# 📚 Protocolos del Ecosistema Yggdrasil

Este directorio contiene los protocolos operacionales del ecosistema. Son las reglas que gobiernan cómo trabajan los agentes IA en cada repo.

## Ficheros

| Fichero | Contenido |
|---|---|
| `inicio-sesion.md` | Protocolo de arranque para cualquier repo |
| `cierre-sesion.md` | Protocolo de cierre y handoff |
| `por-repo/` | Protocolos específicos por repo |

## Regla fundamental

> **Ningún agente empieza a trabajar sin leer `AGENT.md` del repo.  
> Ningún agente termina sin actualizar `CONTEXT.md` y crear issue si hay trabajo pendiente.**
