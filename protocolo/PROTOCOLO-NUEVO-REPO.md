# Protocolo Nuevo Repo

## Al crear un repo nuevo en el ecosistema

1. Añadirlo a la tabla de repos en DEW README y wiki README
2. Crear isla en `WIKI---PERSONAL/wiki/islas/`
3. Crear `README.md`, `now.md`, `inbox/` en el repo nuevo
4. Añadirlo al INDEX de islas con estado `🟡 Nuevo`
5. Abrir issue de auditoría de nombres (#65) si hay renombrado
6. Añadirlo al pipeline de indexación de `local-brain` (#67)

## Naming

- Formato: `yggdrasil-[dominio]` para repos del ecosistema
- Excepción: repos con nombre propio (`WIKI---PERSONAL`, `THDORA-PERSONAL`)
