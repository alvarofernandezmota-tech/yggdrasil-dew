# Protocolo de Consistencia de Nombres

## Al renombrar un repo

1. Renombrar en GitHub Settings
2. Abrir issue de auditoría en DEW (ver #65)
3. Buscar nombre antiguo en toda la org con `search_code`
4. Corregir todas las referencias encontradas
5. Actualizar INDEX.md de la wiki
6. Cerrar el issue de auditoría

## Nombres canónicos actuales

| Nombre canónico | Nombres incorrectos a buscar |
|---|---|
| `yggdrasil-formacion` | `yggdrasil-formacion-` |
| `WIKI---PERSONAL` | `yggdrasil-wiki`, `yggdrasil-wiki-personal` |
| `yggdrasil-tracking` | `VIDAPERSONAL`, `vida-personal` |
