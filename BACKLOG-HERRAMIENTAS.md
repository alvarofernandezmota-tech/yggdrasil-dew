# Backlog de herramientas

## Alta prioridad
- CLI personal para notas rápidas.
- Script de respaldos comprimidos con rotación.
- Generador automático de estructura para READMEs.

## Prioridad media
- Bot de notificaciones locales.
- Clasificador automático de descargas.
- Dashboard minimalista de recursos en terminal.

## Regla
Toda herramienta debe ser simple, documentada, reusable y ejecutable desde terminal.
