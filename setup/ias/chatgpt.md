# ChatGPT — OpenAI GPT-4o
*Última actualización: 15 junio 2026*

## Datos clave

| Campo | Valor |
|-------|-------|
| Acceso | ChatGPT Plus / web |
| Límite | Variable por periodo (se agota con uso intensivo) |
| Reset | Al día siguiente (medianoche hora local) |
| Memoria entre sesiones | ✅ Sí — única IA con memoria nativa persistente |

## Fortalezas

- **Memoria persistente** entre sesiones — recuerda proyectos, nombres, contexto
- Interpreta archivos (PDFs, Excel, CSVs) directamente
- DALL-E integrado (mockups, branding, imágenes de Toki)
- Buen equilibrio velocidad/calidad en conversación
- Búsqueda web actualizada

## Debilidades

- Límite diario de mensajes GPT-4o (se agota rápido con uso intensivo)
- Más complaciente que Claude (valida en lugar de criticar)
- Reset al día siguiente puede cortar sesiones nocturnas

## Mejor uso

- Continuidad de proyectos sin reexplicar contexto
- Análisis de archivos (subir CSVs de métricas, PDFs de estudios)
- Brainstorming creativo y naming
- Generación de imágenes para mockups de Toki
- Tareas que requieren seguimiento de varios días

## Estrategia cuando se agota

- GPT-4o agotado → cambiar a GPT-4o mini (límite más generoso)
- Todo agotado → rotar a Claude o Mistral + documentar en registro
