## Lecciones aprendidas

1. **Imágenes minimalistas Docker** (qdrant, open-webui) no incluyen `wget`/`curl`. Usar TCP bash check como healthcheck universal.
2. **No pegar respuestas de IA directamente en la terminal** — se ejecutan como comandos.
3. Un pull de Ollama atascado visualmente puede haber terminado — verificar siempre con `ollama list`.
4. tmux útil para sesiones largas pero no imprescindible para operativa básica.

## Patrón healthcheck universal (reutilizable)

```bash
--health-cmd "bash -c 'cat < /dev/null > /dev/tcp/localhost/PUERTO'"
--health-interval 30s
--health-timeout 5s
--health-retries 3
```
