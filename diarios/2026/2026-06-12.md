## 📚 Formación — aprendido hoy

- **Tailscale en Arch** — instalación, auth, fix NeedsLogin
- **Input Leap en Wayland** — diferencia binario `input-leaps` vs wrapper `input-leap`
- **libei / InputCapture** — protocolo Wayland; versión estable no lo implementa
- **AUR git builds** — paquetes `-git` adelantan parches que el estable no tiene
- **xdg-desktop-portal-hyprland** — necesario para exponer `InputCapture` al servidor
- **UFW Zero Trust** — deny incoming + whitelist Tailscale
- **IaC personal** — toda config bajo Git

**Concepto clave:**
> *"El paquete estable va detrás. En Wayland, los protocolos nuevos (libei, InputCapture) solo están en las ramas -git. Migrar al AUR git build es la decisión correcta, no un parche temporal."*
