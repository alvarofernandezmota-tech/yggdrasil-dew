## 📚 Formación — aprendido hoy

- **Tailscale en Arch** — `pacman`, `systemctl enable --now`, flujo de auth
- **`--force-reauth`** — regenera el enlace de auth sin reinstalar
- **Input Leap en Wayland** — `QT_QPA_PLATFORM=wayland` necesario en el unit file
- **systemd usuario vs sistema** — `systemctl --user` para servicios de sesión gráfica
- **UFW Zero Trust** — `deny incoming` + whitelist por IP Tailscale
- **libei / InputCapture** — protocolo Wayland para captura de input; Input Leap estable no lo implementa bien
- **AUR git builds** — diferencia entre paquete estable y `-git` (rama dev con parches adelantados)

**Concepto clave del día:**
> *"El paquete estable va detrás. Cuando el problema es de compatibilidad con protocolos nuevos de Wayland, la solución casi siempre está en la rama `-git` del AUR."*
