## 🧠 Aprendizajes

- **MIUI bloquea ADB installs** por defecto aunque ADB esté activo — necesita "Instalar vía USB" en opciones dev
- **Android 13+ bloquea `input keyevent`** sin permiso `INJECT_EVENTS` — no disponible sin root
- **F-Droid es fuente fiable** para APKs de apps open source cuando GitHub bloquea curl
- **Siempre usar tmux** en madre para cualquier proceso largo (docker pull, downloads) — la red cae
- **IP Tailscale de madre** (`100.91.112.32`) solo accesible cuando Tailscale está activo en el cliente
- **IP local de madre**: `10.48.234.18` (interfaz `enp0s20f0u3`)
- Para ver pantalla Redmi sin display: `adb shell screencap -p /sdcard/screen.png && adb pull /sdcard/screen.png ~/screen.png`
