# Herramientas Open Source Control Móvil

**Estado:** 💡 Idea  
**Prioridad:** Baja

## Investigar

- **scrcpy** — control pantalla Android desde PC por USB/WiFi (open source, muy popular)
- **Termux** — terminal Linux en Android, scripting nativo
- **KDE Connect** — integración móvil-PC (notificaciones, archivos, portapapeles)
- **ADB WiFi** — controlar móvil sin cable por red local
- **Gadgetbridge** — si tienes wearables

## Primer paso

```bash
sudo pacman -S scrcpy android-tools
```
