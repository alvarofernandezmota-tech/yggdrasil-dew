### Valoración

**Bien:**
- Default deny incoming — correcto
- Todos los servicios (Ollama, Qdrant, Grafana, Open WebUI, Portainer, n8n) solo accesibles desde `100.64.0.0/10` (rango Tailscale) — perfecto
- fail2ban activo como segunda capa

**Riesgo detectado:**
- **Puerto 22 abierto a `Anywhere`** — SSH expuesto a internet pública
- fail2ban mitiga fuerza bruta, pero la superficie de ataque es innecesaria si todo el acceso SSH es vía Tailscale

### Recomendación (HAL-006 pendiente)

Restringir SSH a solo Tailscale:
```bash
# Añadir regla específica Tailscale
sudo ufw allow in on tailscale0 to any port 22
# Eliminar la regla abierta
sudo ufw delete allow 22/tcp
# Verificar
sudo ufw status verbose
```

**Antes de ejecutar:** verificar que tienes acceso a Madre por Tailscale desde otro dispositivo.
