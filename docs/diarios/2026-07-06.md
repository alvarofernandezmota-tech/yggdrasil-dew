### Punto pendiente — branch protection

**Es el punto más importante de Fase 4** — impide force-push y borrado de historial en `main`.
No ejecutable vía MCP (requiere token con permisos `admin:repo`).
Documentado en [issue #25](https://github.com/alvarofernandezmota-tech/yggdrasil-dew/issues/25) con comandos exactos para ejecutar desde terminal Madre.

```bash
# Ejecutar desde Madre cuando haya terminal
gh api repos/alvarofernandezmota-tech/yggdrasil-dew/branches/main/protection \
  --method PUT \
  --field allow_force_pushes=false \
  --field allow_deletions=false
# Repetir para WIKI---PERSONAL y yggdrasil-secops
```
