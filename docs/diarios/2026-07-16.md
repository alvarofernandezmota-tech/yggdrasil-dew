
## Decisiones tomadas hoy

1. **Arquitectura 3 niveles**: global → por repo → plantillas. Canon desde 2026-07-16.
2. **SINCRONIZACION vs AUDITORIA vs ACTUALIZACION**: tres protocolos distintos con roles distintos — no se mezclan.
3. **CIERRE-SESION como orquestador**: llama a SINCRONIZACION de los repos tocados. El ciclo cierra solo.
4. **Todo va al diario DEW**: el diario DEW registra el trabajo técnico de sesión. El diario TRACKING registra el estado personal del día. Son complementarios, no sustitutos.
5. **Protocolos para partes pequeñas**: pendiente añadir protocolos micro (ADR, isla nueva, apunte, entrada tracking) — se documenta como siguiente paso de F15.
