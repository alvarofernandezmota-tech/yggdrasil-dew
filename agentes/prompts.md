# Prompts — Mantenimiento del repo y sistema

> Prompts probados para las operaciones más frecuentes.
> Copiar, adaptar y usar directamente con el agente.

---

## 🔄 Actualizar CONTEXT.md

```
Actualiza mi CONTEXT.md con la información de esta semana.
Fecha actual: [FECHA]
Proyecto principal esta semana: [PROYECTO]
Avances: [QUÉ AVANCÉ]
Pendientes que pasan: [PENDIENTES]
Formación activa: [QUÉ ESTOY APRENDIENDO]
Cómo me siento con el sistema: [OPCIONAL]
```

---

## 📝 Crear entrada de diario

```
Crea la entrada del diario de hoy usando la plantilla _dia.md.
Fecha: [FECHA]
Cómo ha ido el día: [RESUMEN]
Tareas completadas: [LISTA]
Hábitos: [EJERCICIO/LECTURA/CÓDIGO: ✅ o ❌]
Lo mejor y lo que cambiaría: [OPCIONAL]
```

---

## 📊 Crear cierre semanal

```
Crea el cierre de la semana [N] usando la plantilla _semana.md.
Semana del [FECHA_INICIO] al [FECHA_FIN]
Proyecto principal: [PROYECTO]
¿Se cumplió el objetivo?: [SÍ/NO/PARCIAL]
Hábitos resumen: [DATOS]
Lo más importante: [3 COSAS]
Foco semana siguiente: [PROYECTO + 3 TAREAS]
```

---

## 🆕 Documentar proyecto nuevo

```
Crea la ficha del proyecto [NOMBRE] en proyectos/[nombre].md
Qué es: [DESCRIPCIÓN]
Por qué existe: [MOTIVACIÓN]
Stack: [TECNOLOGÍAS]
Estado actual: [ESTADO]
Próximos pasos: [LISTA]
```

---

## 📅 Abrir nuevo mes

```
Estamos empezando [MES] [AÑO].
1. Crea la carpeta diarios/[AÑO]/[MM]-[mes]/
2. Actualiza CONTEXT.md con los objetivos del nuevo mes
3. Añade una línea al CHANGELOG.md
4. Dime qué objetivos del mes anterior se cumplieron y cuáles pasan
```

---

## 🔍 Auditoría del repo

```
Haz una auditoría rápida del repo personal-v2:
1. ¿Qué archivos no se han actualizado en más de 4 semanas?
2. ¿Hay proyectos en CONTEXT.md que no tienen su archivo en proyectos/?
3. ¿CONTEXT.md refleja la realidad actual?
4. ¿Qué falta documentar?
```
