---

## Cómo usar esto

Cuando necesites recordar cómo funciona algo en Python, buscas aquí.
No vas a `curso/m3/tema-2/...` — vas a `referencia/poo.md` y lo tienes en 10 segundos.

**Regla:** si un concepto aparece más de una vez en el trabajo real, tiene que estar aquí.

## Cómo se construye

1. Terminas un módulo del curso
2. Extraes los conceptos clave a su archivo de referencia
3. Añades ejemplos propios — no copias del curso, sintetizas
4. El archivo queda como chuleta definitiva de ese concepto

_Construyendo a partir de M1–M5 completados._
