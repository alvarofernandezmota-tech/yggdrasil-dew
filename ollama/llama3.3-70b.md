# Llama3.3-70B — Potente, Solo Madre

Modelo grande de Meta. Alto rendimiento pero requiere mucha VRAM.

## Hardware
- VRAM necesaria: +40 GB ❌ Solo Madre con mucha VRAM
- Con offloading a RAM: ~3-5 tok/s

## Cuándo usarlo
- Máximo razonamiento local
- Cuando los modelos pequeños no dan la talla

```bash
ollama pull llama3.3:70b
```
