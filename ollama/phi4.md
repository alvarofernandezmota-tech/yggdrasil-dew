# Phi4 — Modelo Ligero y Rápido

Modelo de Microsoft. Muy ligero, ideal para tareas rápidas con poco contexto.

## Hardware
- VRAM mínima: 4 GB ✔️ Acer + Madre
- Velocidad Madre: ~25 tok/s

## Cuándo usarlo
- Respuestas rápidas sin necesidad de razonamiento profundo
- Cuando la VRAM es limitada

```bash
ollama pull phi4
ollama run phi4
```
