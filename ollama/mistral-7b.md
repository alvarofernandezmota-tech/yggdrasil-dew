# Mistral 7B — General Ligero

Modelo general de Mistral AI. Ligero y rápido, buen punto de entrada.

## Hardware
- VRAM mínima: 5 GB ✔️ Madre (justo en Acer)
- Velocidad Madre: ~20 tok/s

## Cuándo usarlo
- General purpose ligero
- Alternativa a llama3.2 si quieres probar otro modelo

```bash
ollama pull mistral:7b
ollama run mistral:7b
```
