# Gemma3 — General Purpose Local

Modelo de Google, buena relación calidad/tamaño para uso general.

## Hardware
- VRAM mínima: 3-4 GB ✔️ Acer + Madre
- Velocidad Madre: ~22 tok/s

## Cuándo usarlo
- General purpose cuando llama3.2 no es suficiente
- Buen español

```bash
ollama pull gemma3
ollama run gemma3
```
