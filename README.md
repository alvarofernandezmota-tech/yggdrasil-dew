# personal-v2

> Base de conocimiento personal de Álvaro Fernández Mota.
> Funciona como una base de datos en Markdown: cada carpeta es una tabla, cada archivo es un registro, cada sección (`##`) es un campo.

---

## Empieza aquí

```
CONTEXT.md   → qué está pasando ahora mismo
AGENT.md     → cómo navegar el repo y reglas del sistema
filosofia.md → principios técnicos inamovibles
```

---

## Estructura

```
personal-v2/
├── CONTEXT.md          ← estado actual (actualizar cada domingo)
├── AGENT.md            ← instrucciones para el agente IA
├── CHANGELOG.md        ← historial de cambios del repo
├── filosofia.md        ← principios técnicos (actualizar mensual)
├── yo/                 ← perfil, objetivos, empleabilidad
├── proyectos/          ← proyectos activos y pausados
├── formacion/          ← áreas de aprendizaje
├── setup/              ← hardware, infraestructura, servidor
├── diarios/            ← memoria episódica diaria
└── agentes/            ← roles IA, prompts, integraciones
```

---

## Filosofía

- **Minimalista** — solo lo que aporta valor. Sin archivos decorativos.
- **Estructurado** — secciones estándar en cada tipo de archivo.
- **Siempre actualizado** — cada acción queda registrada.
- **Open source** — todo el stack es libre. Sin excepciones.
- **Privado por diseño** — datos sensibles fuera del repo público.

→ Ver detalle completo: `filosofia.md`
